# WSL (Windows)

Linux용 Windows 서브 시스템(WSL)을 사용하면 개발자가 가상 컴퓨터의 오버헤드 없이 Windows에서 직접 CLI 도구, 유틸리티 및 응용 프로그램을 비롯하여 GNU/Linux 환경을 그대로 유지하면서 실행할 수 있습니다.

주의!

WSL을 사용하려면 Windows 10 (v1607) 이상이 요구됩니다. Windows 10 사용자가 아닐 경우 Git Bash를 사용하세요.

# 설치 및 설정

Windows 10에서 WSL 설치 및 설정 방법은 윈도우즈에서 리눅스 설치 - WSL 문서를 참고하세요.
