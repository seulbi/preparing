# Homebrew (Mac)

Mac OS 용 패캐지 매니저 Homebrew를 활용해 필요한 패키지를 설치 관리하세요.

# 설치

Brew 설치

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# 명령

패키지 검색

$ brew search node

패키지 설치

$ brew install node

# 특정 버전의 패키지를 설치하고자 할 경우
brew install node@9

패키지 제거

$ brew uninstall node

설치된 패키지 목록 출력

$ brew list


NOTE

보다 자세한 사용 방법은 Brew 문서를 참고하세요.

참고!

Linux, Windows 10 사용자는 Linuxbrew를 설치해 Homebrew를 사용할 수 있습니다.
