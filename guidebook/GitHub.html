# GitHub 서비스

Git을 사용한 버전 관리 시스템을 온라인 상에서 이용할 수 있도록 서비스 합니다. 무료 또는 유료(추가 기능 제공)로 서비스를 이용할 수 있으며, 전 세계적으로 소스 코드 공유 및 참여(기여)가 활발히 이루어지고 있습니다.

꼭 GitHub 계정을 만들어야 하나?

수업이 아니더라도 GitHub 계정을 만들어 두면 여러모로 쓸모 있습니다.

# Github 가입

  1. Github.com 접속
  2. 화면 우측 상단의 Sign Up 버튼 클릭
  3. Username / Email Address / Password 입력 후, Create an acount 버튼 클릭하여 가입
  4. Personal Plan 선택 후, Finish sign up 버튼 클릭

참고 URL: 그래픽 이미지로 확인하는 Github 가입 절차

# Github 공개된 저장소 둘러보기

  1. Explore Github 접속
  2. 관심있는 분야의 카테고리를 클릭한 후, 저장소를 살펴본다.
  3. 다른 방법은 Search Github 입력 창에 관심사를 입력 후 저장소를 살펴본다.

# Github 저장소(Repository) 생성

  • 저장소 이름 (e.g) https://github.com/{user_id}/<repository_id>)
  • 공개 여부 (※ 비 공개는 유료 버전에서만 가능. 즉, 무료 사용자는 기본 공개 됨)
  • 소개 글 (README.md 파일 추가)
