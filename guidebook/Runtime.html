# Node.js®

Node.js®는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임입니다.

# 설치

Node.js를 여러분의 OS 플랫폼에 설치합니다. 이미 설치한 분들은 LTS로 버전 업그레이드 하세요.

NOTE

Homebrew, Chocolatey와 같은 패키지 매니저를 사용해 설치해도 됩니다.

  • Homebrew : Mac OS 사용자
  • Chocolatey : Windows 사용자

# 버전 관리

Node.js 버전을 다수 관리하고자 한다면 NVM(Node Version Manager)을 사용합니다.
